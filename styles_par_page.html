1. index.html - Page d'Accueil

Balises HTML utilisées :

  • <body> → body
  • <header> → header
  • <h1> → header h1
  • <nav> → nav ul, nav ul li, nav ul li a
  • <main> → main
  • <h2> → main h2
  • <section> → section
  • <p> → section p
  • <strong> → article strong
  • <footer> → footer

Classes CSS utilisées :

  • .bio - Section de présentation bio
  • .profile - Image de profil circulaire
  • .socials - Conteneur des liens sociaux
  • .social-icon - Icônes des réseaux sociaux

Styles CSS appliqués :

body: font-family, margin, padding, background gradient

header: background gradient, color, padding, margin, box-shadow

header h1: text-align, margin, color, font-weight, text-shadow, letter-spacing

nav ul li a: color, text-decoration, padding, display, border-radius, font-weight

main: max-width, margin, padding, background gradient, border-radius, box-shadow

main h2: color, margin, text-align, font-size, font-weight, text-shadow

section: margin, padding, background gradient, border, border-radius, box-shadow

.bio: text-align, padding, background gradient, border-radius, box-shadow, border

.profile: width, height, border-radius, margin, border, box-shadow

footer: background gradient, color, padding, margin, text-align, box-shadow

footer .socials a: background gradient, box-shadow, padding

.social-icon: width, height, display

Styles Communs à Toutes les Pages

Structure commune :

  • body: font-family, margin, padding, background gradient
  • header: background gradient, color, padding, margin, box-shadow
  • header h1: text-align, margin, color, font-weight, text-shadow, letter-spacing
  • nav ul: list-style, padding, margin, text-align
  • nav ul li: display inline-block, margin
  • nav ul li a: color, text-decoration, padding, display, border-radius, font-weight
  • nav ul li a:hover: background gradient, color, box-shadow, border-bottom
  • main: max-width, margin, padding, background gradient, border-radius, box-shadow
  • main h2: color, margin, text-align, font-size, font-weight, text-shadow
  • footer: background gradient, color, padding, margin, text-align, box-shadow
  • footer .socials a: background gradient, box-shadow, padding
  • .social-icon: width, height, display
  • footer p: margin, color, font-size

Media Queries (Responsive) :

@media (max-width: 768px):

  • header h1: font-size réduit
  • nav ul li: display block
  • main h2: font-size réduit
  • section, article: padding réduit
  • .profile: width et height réduits
  • .contact-cards: grid-template-columns 1fr
